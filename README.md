## Summary

The PES is the numeric metric expressing containment performance of a security tool or autonomous agent measured under the Proof Protocol.

**Formula:** `PES = Blocked / (Blocked + Missed) × 100`

OBSERVED and IRRELEVANT cases are explicitly excluded from the denominator. A score without a declared denominator is not a metric.
